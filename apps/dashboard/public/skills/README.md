# Skill icons

27 of the 29 RS3 skill icons, named `<skill>.png` (lowercase), used by the
skills panel. Sourced from the RuneScape Fandom wiki CDN.

**Missing:** `archaeology.png` and `necromancy.png` — the Fandom CDN didn't have
them under a predictable name. The panel falls back to a clean 3-letter tile
("ARC", "NEC") when an icon file is absent, so nothing looks broken. To add them,
drop correctly-named PNGs in this folder (grab them from the in-game stats panel
or runescape.wiki) and they'll appear automatically — no code change needed.
